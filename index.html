【Linux学习】②文件权限

发表于 2022-04-13

1、Linux文件系统

Linux 系统是一种典型的 多用户系统 ，不同的用户处于不同的地位，拥有不同的权限。为了保护系统的安全性，Linux 系统对不同的用户访问同一文件（包括目录文件）的权限做了不同的规定。

Linux下文件的权限类型一般包括

  • 读(r)
  • 写(w)
  • 执行(x)

2、查看文件权限

使用 ls -l 或者 ll 命令可以查看文件权限




d 在 Linux 中代表该文件是一个目录文件。
在 Linux 中第一个字符代表这个文件是目录、文件或链接文件等等，常用的还有

  • d 目录
  • - 文件
  • l 链接文档
  • b 装置文件里面的可供储存的接口设备
  • c 装置文件里面的串行端口设备，例如键盘、鼠标

3、使用chmod来修改文件的权限

在用户主目录下，创建名命为shell-226的目录,并在此目录下创建名为 jiaoben-226.sh 的脚本文件

使用nano编辑器写入内容：

  • 将脚本权限修改为任何人都不可以执行
    chmod 000 jiaoben-226.sh
    ls -l查看文件权限
    这时执行脚本是无法执行的

  • 只有root用户或者使用sudo可以直接执行

  • 再将脚本权限修改为任何人都可以执行
    chmod 777 jiaoben-226.sh
    ls -l查看文件权限
    这时执行脚本正常执行

  • 解释一下000、777、775这种数字是怎么来的：
    Linux权限有一个分数的概念，r:4 w:2 x:1所以 775=[4+2+1][4+2+1][4+0+1] 其他的也可以以此类推

【Linux学习】①VMware安装Ubuntu20.04

发表于 2022-04-13

1、准备工作

1.1、一台性能较好的主机

我安装在自己的笔记本上配置如下：

  • CPU: i7-9750H
  • 内存: 16G
  • 硬盘: 512G固态硬盘

1.2、VMware虚拟化软件

VMwarre官网下载：https://customerconnect.vmware.com/cn/downloads/#all_products
选择这个VMware Workstation Pro

1.3、Ubuntu镜像

  • 可以去Ubuntu官网的下载界面查看主机是否满足最低配置，这里选择LTS版本，更稳定。

2、开始安装

2.1、安装设置

  • 打开VMware选择新建虚拟机,选择创建典型虚拟机,选择准备工作中下载好的 .iso 镜像

  • 选择之前下载好的Ubuntu镜像，一般这时候VMware会自动识别你的镜像,这时候可以选择简易安装，省去一些麻烦

  • 设置自定义的计算机名字和密码

  • 选择一个合适的路径来安装这台虚拟机（安装在固态硬盘里开机较快）

  • 选择一个合适的硬盘容量（建议将虚拟磁盘存储为单个文件，这样比较好管理)

  • 到这可以自定义虚拟机的硬件配置或者使用推荐配置，这里我多加了两个CPU

  • 点击完成后开始安装，一般十分钟左右可以安装完成，视网络情况好坏

2.2、时区设置

  • 在设置里找到 Date&Time 进入 Time Zone
  • 设置为上海时间

2.3、语言设置

  • 在设置里找到语言设置，安装简体中文后把汉语移动到最顶端后，最后重启系统
  • 重新开机后系统会提示要不要更改用户目录文件夹的名称，建议不要修改，保留旧的名称

2.4、root密码设置

  • 使用 sudo -i 或者 sudo root 进入root用户模式

3、安装VMware Tools

  • VMware Tools:只有在VMware虚拟机中安装好了VMware Tools，才能实现主机与虚拟机之间的文件共享，同时可支持自由拖拽的功能，鼠标也可在虚拟机与主机之间自由移动（不用再按ctrl+alt），且虚拟机屏幕也可实现全屏化

3.1、设置重新安装

  • 在虚拟机标签上右键选择重新安装VMware Tools，不知道是系统的bug还是VMware的bug，如果不重新安装创建的共享文件夹不能同步，所以这里重装一次。安装的过程大概是VMware会将VMware Tools载入到虚拟机的虚拟光驱中，这样在虚拟机里就可以直接安装了
  • 如果重新安装是灰色的，就要在设置里将虚拟CD 已连接 和 启动时连接 打开

3.2、系统里安装

  • 在虚拟机里打开CD，将VMwareTools这个压缩包复制到你的用户目录下
  • 选择文件解压到当前目录（Linux里的中文叫提取）
  • 进入终端，进入到解压出来的目录中执行 然后一路回车，使用默认选项安装即可

3.3、创建共享文件夹

  • 首先在设置里面把这个共享文件夹设置成总是启用
  • 点击添加，在主机里选择一个文件夹作为共享的文件夹
  • 操作完记得点击保存

3.4、测试共享文件夹

  • 在虚拟机的 /mnt/hgfs/ 目录下面就是上一步中创建的共享文件夹

  • 使用nano编辑器在这里创建一个文件进行测试
  • 写入一些内容：学习ubuntu 20.04，学号：226
  • 最后在主机的文件夹中查看，有修改过的文件表示共享文件夹创建成功
