【Linux学习】⑤环境变量和gcc编译器

发表于 2022-04-20 更新于 2022-04-21

gcc编译器

  • gcc的参数和基本用法 https://www.runoob.com/w3cnote/gcc-parameter-detail.html

    使用 gcc -v 查看系统是否有gcc编译器
    没有则使用 sudo apt install gcc 安装

编译hello程序

  • 在主目录下建立工作文件夹,我这里是 /home/xxt/program
    编译后的二进制文件放在 /home/xxt/program/bin

  • 新建文件hello.c并写入打印内容

  • 保存后使用gcc ./hello-226.c -o hello编译，在program目录下会生成可执行文件hello,直接输入hello可以看到成功打印出了写入信息

编译caculat程序

  • 同上建立一个caculat.c文件

  • 写入内容

  • 保存后使用gcc ./caculat-226.c -o caculat-226编译，在program目录下会生成可执行文件

临时添加环境变量

  • 使用export申明
  • 注销或重启后无法再使用声明过的环境变量

当前用户的全局设置

  • 编辑用户主目录下的.bashrc文件，在此文件首或尾添加：
  • 使用source ~/.bashrc更新环境变量
